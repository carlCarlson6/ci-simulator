# Cyberpunk Terminal Simulator

A full-screen terminal simulator built with **React**, **Vite**, **TanStack Start**, and **TypeScript**. Features a cyberpunk/Matrix-inspired UI with neon green-on-black aesthetics, a fully simulated in-memory file system, and type-safe server functions for real backend capabilities.

## рџЋЇ Project Scope

**Goal:** Build a browser-based terminal that feels like a real Unix shell, with a striking cyberpunk visual identity.

**Core Features:**
- Full-screen terminal UI with neon green-on-black color scheme
- In-memory file system (flat Map structure) with pre-populated directories and files
- 16 implemented commands with color-coded output
- Command history (up/down arrows) вЂ” session-only, not persisted
- Tab completion for commands and file paths
- Server API middleware ready for real system info (`whoami`, `hostname`, `date`)
- Error handling with red neon error messages
- Easter egg (`rm -rf /`)

**Out of Scope (MVP):**
- File persistence across reloads
- Pipes and redirects (`|`, `>`, `>>`)
- Background jobs
- Multi-tab support
- Real OS command execution (file system is simulated; `curl` uses server functions)

**Future Enhancements:**
- `theme` command for color scheme switching
- Terminal games (`snake`, `2048`, `tetris`)
- `sudo` easter egg
- File persistence (localStorage or DB)
- Environment variables (`export`, `env`)
- Network commands (`ping`, `curl` вЂ” simulated)
- ASCII art generators (`cowsay`, `figlet`)
- Multi-tab terminal sessions
- Sound effects

## рџЏ—пёЏ Architecture

- **Frontend:** Vite + React + TypeScript + Tailwind CSS v4 + TanStack Start (SPA mode)
- **State Management:** Zustand (terminal session state)
- **File System:** In-memory flat Map with path keys
- **Server:** TanStack Start server functions (`createServerFn`) вЂ” type-safe RPC, no custom middleware
- **Styling:** Tailwind CSS with custom cyberpunk color tokens
